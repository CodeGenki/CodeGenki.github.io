Work Experience

The past two summers, I've had the opportunity to gain work experience related to my field of interest.

Daikin Industries, Ltd. (Summer 2018)

At Daikin, I worked with a team to build a service solution using IoT and AR technology. I got to develop a web application, AWS platform, and Microsoft HoloLens application. For 3 weeks, I also traveled to Japan to work with the marketing and global operations groups at the headquarters. I got to brainstorm and develop a business model for some new AI related ideas for an upcoming Daikin product.

Shipcom Wireless, Inc. (Summer 2017)

At Shipcom, I was given the task of researching and developing a more modern system of authentication for their supply chain management software, as well as adding useful functionality to this software. I used Microsoft's C# libraries along with other frameworks to build the back-end software. I set up a self-hosted server with support for Web API, file serving, and websockets (using the publisher/subscriber pattern).
